> **Note:** Due to the April 30 (Liberation Day) and May 01 (International Workers' Day) public holidays, this week's schedule is consolidated into 3 working days.

### Weekly Topic

Reviewing AWS technical documentation, mastering the AWS Console, and managing EC2 compute resources.

### Weekly Goals

* In-depth study of operational mechanisms and practical use cases for AWS cloud services.
* Gain proficiency with the AWS Management Console and compute resource management (EC2).
* Analyze secure access patterns from EC2 to other storage services without hardcoding static access keys.

### Work Schedule

| Date | Day | Task Description | Lab / Project |
|---|---|---|---|
| 11/05/2026 | Mon | Review VPC topology and IAM configurations from Week 1. Set up a $5 Billing Alarm to proactively monitor extra costs. | AWS Infrastructure |
| 12/05/2026 | Tue | Study IAM Roles for EC2 and the Instance Profile architecture to secure access while avoiding static access keys. Complete Lab 000048. | [Lab 000048 - IAM Roles cho EC2 (Instance Profile)](https://000048.awsstudygroup.com)|
| 13/05/2026 | Wed | Explore Amazon EC2 compute services. Provision Amazon Linux & Windows instances, configure key pairs, AMIs, and Security Groups. Hands-on practice with Lab 000004. | [Lab 000004 - Giới thiệu Amazon EC2](https://000004.awsstudygroup.com)|

### Expected Results

* Understand how to apply Instance Profiles for secure communication between EC2 and S3 or DynamoDB.
* Master resource management and monitoring capabilities directly on the AWS Console interface.
* Successfully launch, configure, and connect to Linux virtual instances (via SSH/Session Manager) and Windows instances (via RDP).
* Differentiate core concepts including AMIs, Instance Types, Security Groups, and Key Pairs within EC2.
