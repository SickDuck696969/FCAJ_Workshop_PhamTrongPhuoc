### Weekly Topic

Overview of AWS as an enterprise and introduction to core cloud services.

### Weekly Goals

* Get familiar with the direction and framework of the First Cloud AI Journey internship program.
* Gain a fundamental understanding of AWS Vietnam's operational model and workflow.
* Study AWS Core services (IAM & VPC), initializing baseline security configurations and infrastructure.

### Work Schedule

| Date | Day | Task Description | Lab / Project |
|---|---|---|---|
| 04/05/2026 | Mon | Review and align with the general regulations and guidelines of the FCAJ program. | Learning Path |
| 05/05/2026 | Tue | Explore an overview of the AWS ecosystem. Create a personal AWS account and enable Multi-Factor Authentication (MFA) for the root user. Complete Lab 000001. | [Lab 000001 - Creating your first AWS account](https://000001.awsstudygroup.com)|
| 06/05/2026 | Wed | Study IAM in-depth: managing users, groups, and access policies. Apply the Least Privilege principle to harden security. Hands-on practice with Lab 000002 (Part 1.1 - 2.3). | [Lab 000002 - Access Management with AWS IAM](https://000002.awsstudygroup.com)|
| 07/05/2026 | Thu | Differentiate between IAM Users and IAM Roles. Practice the Switch Role operation on the AWS Management Console. Continue Lab 000002 (Part 3.1 - 5). | [Lab 000002 - Access Management with AWS IAM](https://000002.awsstudygroup.com)|
| 08/05/2026 | Fri | Begin exploring Amazon VPC networking services, designing Public/Private Subnet architectures, and configuring basic routing. Hands-on practice with Lab 000003. | [Lab 000003 - Basic Networking with Amazon VPC](https://000003.awsstudygroup.com)|

### Expected Results

* Successfully register an AWS account and secure the root account with MFA.
* Master setting up budget alerts via AWS Budgets to effectively manage costs within the Free Tier limits.
* Implement basic access control using IAM Users and Groups, demonstrating a solid understanding of the Principle of Least Privilege.
* Successfully build a standard VPC environment featuring Public/Private Subnets and a Route Table with internet connectivity.
