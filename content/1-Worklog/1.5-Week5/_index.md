### Week Topic

Study serverless backend design for the game API, focusing on Lambda, API Gateway, and CloudFormation.

### Week Objectives

* Understand the serverless architecture pattern used by the game backend API.
* Configure API resources and integration points for API Gateway.
* Research cost-optimization and infrastructure automation using Lambda and CloudFormation.
* Attend tech networking events at the AWS Vietnam office.

### Expected Outcomes

* Define a clear API Gateway and backend path for the game API workflow.
* Deploy a Lambda Python function to auto-manage EC2 compute state, reducing unnecessary charges.
* Configure a REST API on API Gateway with CORS enabled.
* Write and provision resources using a basic CloudFormation template.
* Engage with local AWS experts at AWS Community Day.
