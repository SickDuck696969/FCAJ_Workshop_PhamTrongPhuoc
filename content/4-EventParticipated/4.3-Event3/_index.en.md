# Summary Report: "FCAJ Event Meet Up"

| Info | Details |
|------|---------|
| Date | 06/06/2026 |
| Location | 26th Floor, Bitexco Tower, 02 Hai Trieu Street, Saigon Ward, Ho Chi Minh City |
| Role | Attend |

### Event Objectives
- Share knowledge and practical experiences in Cloud Computing and AWS ecosystems.
- Explore containerization technologies, specifically Docker, as a foundational skill.
- Understand how to leverage AWS WAF with Machine Learning for proactive Cyber Attack Detection.
- Learn to build scalable game backend solutions using Godot and AWS WebSockets.
- Discover how to build a Graph Knowledge Base for GraphRAG using AWS Neptune.
- Provide career progression guidance for transitioning from IT Helpdesk to Cloud/DevOps roles.
- Emphasize the importance of effective teamwork and collaboration skills in technical projects.

### Speakers
- **Mr. Bao Huynh** (Junior Cloud Native Developer, Endava Vietnam & Founder, ITea Lab) - Docker: A containerization technology.
- **Mr. Le Hoang Gia Dai** - Combining AWS WAF with Machine Learning for Cyber Attack Detection.
- **Mr. Nguyen Quoc Bao** - Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets.
- **Mr. Truong Phuoc** - Effective Teamwork.
- **Mr. Viet Phat** - AWS Neptune for Building a Graph Knowledge Base for GraphRAG.
- **Mr. Vinh Tran** (System Administrator, Central Retail Group) - From IT Helpdesk to Senior Sysadmin and Cloud/DevOps.

### Key Highlights

#### Containerization & Docker
- Gained foundational understanding of Docker as a core containerization technology, crucial for modern application deployment.

#### Cyber Security with Machine Learning
- Explored the integration of AWS WAF (Web Application Firewall) with Machine Learning models to create an intelligent Network Intrusion Detection System (NIDS) for robust cyber attack prevention.

#### Cloud Game Backend
- Learned architectural patterns for connecting game clients (built with Godot) to AWS WebSockets to enable real-time, multiplayer gaming experiences in the cloud.

#### Graph Databases & GraphRAG
- Deep dived into AWS Neptune and its application in building sophisticated Graph Knowledge Bases, which are essential for powering GraphRAG (Retrieval-Augmented Generation) applications.

#### Career Roadmap & Soft Skills
- Received practical guidance and roadmap strategies for self-studying and transitioning from an IT Helpdesk role to a Senior Sysadmin and eventually into Cloud/DevOps engineering.
- Discussed key frameworks and mindsets required for effective teamwork in fast-paced tech environments.

### Applying to Work
- **Containerization:** Start containerizing local development environments and personal projects using Docker.
- **Security Awareness:** Consider implementing basic AWS WAF rules for personal web projects to understand cloud security mechanisms.
- **Career Planning:** Structure a clear learning path toward Cloud/DevOps based on the shared roadmap, focusing on essential self-study milestones.
- **Team Collaboration:** Apply the discussed effective teamwork principles in current group projects and academic assignments.
